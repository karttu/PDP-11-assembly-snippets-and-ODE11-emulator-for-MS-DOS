miscellaneous PDP-11 related stuff

Here, by me

  • GLAD.LSP is a beginning of a simple Generic Lisp Assembler/Disassembler written with St. Vitus Lisp
    It has a simple implementation for PDP-11 instruction set in a separate file PDP11SET.LSP
  • REGPER.MAC is a geeky little program for RT-11 that can be used to make a working copy of itself, with all the register fields in its instructions appropriately permuted.
  • REGCOP.MAC is a similar piece of code for ODE-11 PDP-11 emulator for MS-DOS.
